# Remove-AzsPlan

## SYNOPSIS
Removes the specified plan

## SYNTAX

### Plans_Delete (Default)

```

### ResourceId_Plans_Delete
```
Remove-AzsPlan -ResourceId <String> [-Force] [<CommonParameters>]
